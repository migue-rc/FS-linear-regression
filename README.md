# Linear Regression From Scratch

This project provides an educational, deep dive explotarion of **simple linear regression**, 
built entirely from scratch using Python and Numpy.
It is designed to bridge the gap between high level machine learning concepts and the fundamental mathematics that power them.

## Overview

The core of this project is an interactive [quarto notebook](https://migue-rc.github.io/FS-linear-regression/) that guided you through the entire lifecycle on a *Linear Regression* model.
Rather than relying on high level libraries like scikit-learn, I implemented the optimization process manually to provide a transparent look at how machines actually "learn".

## Educational Focus

This serves as a foundational resource for anyone looking to understand:
- The relationship between independent and dependent variables.
- How a loss function quantifies model performance.
- The iterative nature of optimization algorithms.
- How to visualize data to gain insights into model convergence and stagnation.

## Key Features

* **Mathematical Derivations**
* **Gradient Descent Intuition**
* **Interactive Visualizations**
